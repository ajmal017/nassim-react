# Kurtosis React Frontend

## Stack
- Frontend: React, Redux
- Styling: Bootstrap, SCSS, Sass
- Authentication: Google Sign-in, Facebook Sign-in
- Backend: Node
- Database: MongoDB Atlas, Google Cloud
- Deployment: Heroku, Netlify, Travis CI
- Testing: Enzyme
